# Cloned Repositories

## 1. Entity Tracking in Language Models
- **URL**: https://github.com/najoungkim/code-models-entity-tracking
- **Purpose**: Official implementation for Kim & Schuster (2023). Contains code for generating state-tracking datasets and evaluating models.
- **Location**: `code/entity-tracking/`
- **Key Files**:
    - `data_generation/`: Scripts to create the tracking tasks.
    - `evaluation/`: Scripts to evaluate model performance.

## 2. ETHIC Benchmark
- **URL**: https://github.com/dmis-lab/ETHIC
- **Purpose**: Benchmark for evaluating LLMs on long-context tasks with high information coverage.
- **Location**: `code/ETHIC/`
- **Key Files**:
    - `data/`: Contains the benchmark data (or scripts to download it).
    - `eval/`: Evaluation scripts.

## Notes
These repositories provide the necessary tools to:
1.  Generate synthetic entity tracking tasks (from Kim & Schuster).
2.  Evaluate on a rigorous long-context benchmark (ETHIC).
