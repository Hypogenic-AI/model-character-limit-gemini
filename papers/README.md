# Downloaded Papers

1. **[Entity Tracking in Language Models](2305.02363_Entity_Tracking_Kim_Schuster.pdf)**
   - **Authors**: Najoung Kim, Sebastian Schuster (2023)
   - **arXiv**: 2305.02363
   - **Relevance**: Investigates LLMs' ability to track entity states. Finds that code-pretraining helps.

2. **[Code Pretraining Improves Entity Tracking Abilities of Language Models](2405.21068_Code_Pretraining_Entity_Tracking.pdf)**
   - **Authors**: (2024)
   - **arXiv**: 2405.21068
   - **Relevance**: Systematically examines the hypothesis that code pretraining enhances discourse entity tracking.

3. **[SCORE: Story Coherence and Retrieval Enhancement for AI Narratives](2503.23512_SCORE_Story_Coherence.pdf)**
   - **Authors**: Qiang Yi et al. (2025)
   - **arXiv**: 2503.23512
   - **Relevance**: Introduces a framework for dynamic state tracking to resolve narrative inconsistencies.

4. **[ETHIC: Evaluating Large Language Models on Long-Context Tasks with High Information Coverage](2410.16848_ETHIC_Benchmark.pdf)**
   - **Authors**: Taewhoo Lee et al. (2024)
   - **arXiv**: 2410.16848
   - **Relevance**: Benchmark for long-context tasks, highlighting performance degradation in high information coverage scenarios.

5. **[Evaluating Character Understanding of Large Language Models via Character Profiling from Fictional Works](2404.12726_Character_Understanding_Profiling.pdf)**
   - **Authors**: Xinfeng Yuan et al. (2024)

6. **[If an LLM Were a Character, Would It Know Its Own Story? Evaluating Lifelong Learning in LLMs](2503.23514_If_LLM_Were_Character.pdf)**
